## Quick start
1. Clone the repository: `git clone <repo-url> && cd datapipeline-docker-public`.
2. Create your environment file: `cp .env.example .env` then adjust the PostgreSQL credentials if needed.
3. Launch the infrastructure: `docker compose up -d`.
4. Inspect the logs (optional) to confirm PostgreSQL is ready: `docker compose logs -f db`.
